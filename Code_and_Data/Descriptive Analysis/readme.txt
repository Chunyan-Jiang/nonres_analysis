# Descriptive Analysis

This folder contains the descriptive analysis for unit nonresponse and item nonresponse.


## Overview of files

- `des_item_nonresponse.Rmd`  
  Descriptive analysis for item nonresponse.

- `des_unit_nonresponse.Rmd`  
  Descriptive analysis for item nonresponse.

- outputs folder: Results based on test data

## Execution order

No specific execution order is required.

## Inputs and outputs

The scripts assume that cleaned input data are available prior to execution.
All generated figures, tables, and intermediate results are written to subfolders within the `outputs/` directory located in this folder.
