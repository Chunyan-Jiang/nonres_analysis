# Dataset
This folder contains the data for analysis.


## Overview of files

- `data.dta`  
  Original Data

- `unit_nonresponse_data.dta`  
  Cleaned data for descriptive and quantitive analysis for unit nonresponse

- `item_nonresponse_data.dta`  
  Cleaned data for descriptive analysis for item nonresponse

- `imputed_data.dta`  
  Cleaned data for imputation
